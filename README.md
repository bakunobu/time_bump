# time_bump
A convinient timetracker project:
Goals:
- must be minimalistic;
- easy daythrought worktime tracking;
- simple GUI using tk;
- task management;
- save data to a DB;
- interaction with an API (create, add, manage).


Design:
- single window
- two fields;
- one picture;
- three buttons.


Functions:
- start countdown;
- pause countdown;
- bump countdown to the file (date added automatically)
